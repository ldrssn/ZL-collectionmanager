# New Year, New Features: Update Notifications

Communication is key to a good product experience. Today, I implemented a simple but effective notification system to tell users when we’ve added something new.

### Closing the Loop
Whenever we push an update, a "What's New" banner will now appear. It’s a great way to guide users toward new features without them having to hunt for them.

### Precision Stats
On the technical side, I’ve also updated our statistics engine to include the new "Mini" item shapes. It’s important that our **Financial Overview** remains 100% accurate as the physical product line evolves. Accuracy in data leads to trust in the tool.
