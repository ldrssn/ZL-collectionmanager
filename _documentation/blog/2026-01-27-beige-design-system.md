# Designing for Curation: The Beige System

A visual collection manager should be a stage, not the lead actor. Today, I’ve completely refactored the design system to prioritize the items themselves.

### The Brand-Beige Palette
We’ve moved to a high-contrast, **Minimalist** aesthetic centered around `#EFEBE5`. This beige background provides "Visual Silence"—it’s professional and premium, but it stays out of the way of the colorful collection photography.

### The "Perfect Square" Editor
The highlight of today is the new image editor. I’ve built a tool that ensures every photo—no matter how it was taken—can be cropped into a perfect, uniform square. By adding a "Center" shortcut, I’ve made data entry twice as fast while keeping the collection grid looking mathematically perfect.
